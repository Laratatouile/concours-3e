I) La blibliothèque turtle

1) Première étape

première étape: importer la blibliothèque turtle.
C'est la blibliothèque qui permet de faire des dessins de base.
Cette blibliothèque est également assez simple d'utilisation. 

                            
                                from turtle import *

2) Les commandes de base

Turtle permet de dessiner sur un tableau blanc à l'aide d'une flèche applée tortue (turtle en anglais).
Nous disposons de quelques commandes pour diriger la tortue sur l'ecran.


  • Pour avancer de x pixels:

    • (Oui ici on parle maintenant en pixels pour simplifier sur les nombreux apprreils)
    x peut être nimporte quelle valeur même des valeurs négatives ce qui fera reculer la tortue. 

                                    
                                    forward(x)

  • Pour tourner de x degrés:
    • Vers la droite 
                                    
                                    left(x)
    Vers la gauche 
                                    
                                    right(x)

  • Le stylo
    • Nous pouvons aussi choisir de lever ou de descendre le stylo ce qui aura pour effet d'écrire ou non.
    Monter le stylo 
                                    
                                    penup()
    Descendre le stylo 
                                    
                                    pendown()
